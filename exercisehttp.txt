What is HTTP?
    Stands for Hper Tyext Transfer Protocol
    It is for how users get data, send data to a server

What is URL?
    Uniform Resource Locator
    Address for an internet resource

What is DNS? 
    Domain Name System
    system that takes URLS and transforms them to IP addresses

What is a query string?
    Allows user to pass key-value pairs into the URL

2 HTTP Verbs and their uses:
    GET - Get come data
    POST - Send Data

What is a HTTP request?
    Request from a user to a server following HTTP protocol

What is a HTTP response?  
    Response from server to a user follwing HTTP protocol

What is a HTTP header? Give examples of request and response headers you have seen.Provide additional information about the request or the response.
    Headers provide additional information about a request or response.

    Request headers: Host, Accept, Cookie,
    Response headers: Content-Type, Set-Cookie, Cache-Control

What happens when you type a URL in a browser?
    1. Using DNS the name of the URL is turned into an IP address
    2. A request is sent to the IP address
    3. Server sends a response with a status code (200 if good)
    4. Browser makes the DOM derived from the HTML, and also gets any other resources needed (CSS, Images, JavaScript, etc...)
    5. Browser Makes separate HTTP requests for those extra resources and recieves response from the server for each
